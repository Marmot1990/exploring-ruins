# 《探寻遗迹》全文大纲（20章扩展版）

## 整体架构

**核心主题：** 权力不对等下的身体交易，从被迫服从到主动渴求的堕落过程
**风格基调：** 灰暗冷峻，无心理描写，通过动作、对话、环境描写推进
**SM元素贯穿：** 身体强制、权力压迫、疼痛快感交织、角色扮演、公开场合隐秘控制、危险环境禁忌快感
**新增元素：** 
- 被迫与变异生物交配场景
- 多人参与性爱（挑逗剧情）

**总字数：** 约60,000字（20章 × 3000字）

## 人物动态演变（20章完整线）

**沈墨：**
```
被迫契约 → 被迫服从 → 称呼"主人" → 服从内化 → 更深羁绊
→ 角色扮演接受 → 生死臣服 → 公开暴露接受 → 多人场景适应
→ 物品化接受 → 共享服从 → 复杂反应 → 深度依赖
→ 情绪崩溃 → 信任重建 → 主动渴求 → 完全臣服 → 永恒契约
```

**陆骁：**
```
强势建立者 → 控制升级 → 完全支配 → 肾上腺素释放者
→ 公开控制者 → 角色扮演主导 → 生死支配者 → 占有欲宣示
→ 挑逗展示者 → 强迫变异交配 → 物品化极致 → 共享指挥者
→ 惩罚者 → 囚禁调教者 → 安慰者 → 信任重建者
→ 温柔控制者 → 契约确立者 → 永恒主人
```
